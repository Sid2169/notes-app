# Notes App

A simple full-stack note-taking application, built as a learning project.

## Stack

- **Frontend:** Vanilla JavaScript (ES Modules), HTML, CSS, bundled with Webpack
- **Backend:** Node.js + Express.js
- **Database:** SQLite
- **Repository layout:** npm workspaces monorepo

## Project structure

```
notes-app/
├── frontend/   # Browser-side code, bundled by Webpack
├── backend/    # Express server and SQLite access
├── shared/     # Code (mostly constants) used by both frontend and backend
├── package.json
└── README.md
```
